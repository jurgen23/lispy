# Lispy
*mini LISP interpreter*

created using http://www.buildyourownlisp.com/

### Commands
```bash
# build
> make build

# run REPL
> ./lispy

# build with memory check
> make memcheck

# run file
> ./lispy examples/hello_world.lispy
"Hello, World!"
```

### REPL example
```
Lispy Version 0.0.0.0.1
Press Ctrl+c to Exit

lispy> + 1 2 (/ 9 3)
6
lispy> + 3 (% 10 3)
4
lispy>
```
